=========

A small library that adds commas to numbers

## Installation

  `npm install @ffadilaputra/number-formater`

## Usage

    //Import the libraries

    let format = require('@ffadilaputra/number-formatter');

    //Implement

    let output = format(35666);

    // Show the output

    console.log(output)
